Colors in coding:

There are alot of different ways to create colors in coding. we are gonna be covering only the three main ones.

Built in colors

Some programs have built in colors.

At the top of the page there are an input to change the colors of the box.

Try changing the color of the box by typing in: "green" and press ENTER

There are a bunch of other colors like: yellow,red,blue,grey,black,purple,pink,orange,aqua ect.

RGB colors

There are RGB colors. As for instance:rgb(0,255,0).This would make green.

RGB stands for RED,GREEN,BLUE. In RGB you can choose how much of each color shows on each pixel.you can choose values between 0-255.

Try changing the color of the box by typing in: "rgb(255,0,0)" and press ENTER

Then change the numbers to other numbers between 0-255. eg: rgb(56,89,67) or rgb(0,0,255) or rgb(23,255,89)

Numbers lower than 0 or higher than 255 are invalid numbers.

hexadecimal numbers

Then There are hexadecimal values which can generate colors

hexadecimal values usually look like this: #6adb3d,#000,#ff0000 (you can paste this into the input);

hexadecimal vlaues are really complicated, we recommend this website if you want to learn about it:
